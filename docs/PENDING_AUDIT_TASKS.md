## 1. Контент: расширение учебного материала

- [ ] Добавить новые темы **A1/A2** (всего ~10 уроков): «Алфавит и звуки», «Я и мои данные», «Дни недели и месяцы», «Цвета», «Глагол `болу` / нулевая копула», «Мой дом», «Мой город», «Транспорт», «Здоровье и врач», «Погода и времена года».
- [ ] Темы **B1+** (≥10): работа и собеседование, путешествия, аренда, экстренные службы, деловые переговоры, презентация, дискуссия с аргументацией, прямая/косвенная речь.
- [ ] Темы **C1/C2** (≥4): литературный казахский (Абай, Ауэзов), идиоматика, архаичные/поэтические формы, подготовка к ҚАЗТЕСТ C-уровня.
- [ ] Аудио на словарь / диалоги во всех уроках (TTS уже работает; нужно подключить кнопки 🔊 в UI урока).

### 1.2. Раздел грамматики (`/learn/basics`)
- [ ] Добавить недостающие правила в `kazakh-grammar-rules.json`:
  - `rule_26+` — глагольная связка `болу` (A2)
  - `rule_27+` — условное `-са/-се` отдельно с примерами (A2)
  - `rule_28+` — сложные предложения с `-ғанда`, `-ғаныммен`, `қой/ма/ғой` (B1)
  - `rule_29+` — прямая и косвенная речь через `деп / дегенді` (B2)
  - `rule_30+` — залоги (страдательный `-ыл/-іл`, возвратный `-ын/-ін`, взаимно-совместный `-ыс/-іс`) (B2)
  - `rule_31+` — словообразование расширенное (`-шы/-ші`, `-лық/-лік`, `-дай/-дей`, `-гер/-кер`) (B1/B2)
- [ ] C1/C2 уровень: стилистика, регистры, литературный казахский.
- [ ] Аудио-кнопки рядом с примерами (особенно для `rule_18` «специфические звуки»).
- [ ] Картинки/анимация артикуляции для қ, ғ, ң, ө, ү (в правиле rule_18).

## Приоритеты для следующего спринта

Если выбирать топ-5 для следующей итерации (1 спринт = 2 недели), я бы взял:

1. **Контент уроков** — заполнить `LESSON_CONTENT` для L3–L21. Один методист на 5–7 дней.
2. **Аудио-банк для listening в КАЗТЕСТ** и тесте уровня — ~30 коротких фраз с TTS (можно автогенерировать через `/api/learn/tts`).
3. **Reading на казахском** в КАЗТЕСТ — 8–10 текстов 100–200 слов с вопросами.
4. **Ежедневные миссии + Streak Freeze** — закроет retention-loop в геймификации.
5. **Sentry + cost monitoring** — критично для прод-стабильности перед запуском на стабильном домене.

После — расширение тем диалогов, прогресс достижений по всем CEFR, лиги в лидерборде.
