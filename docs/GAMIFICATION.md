# Gamification System

## XP Rewards

| Action | XP |
|--------|-----|
| Lesson complete | 50 |
| Test complete | 30 |
| Perfect test (90%+) | 100 |
| Dialog session | 25 |
| Writing check | 20 |
| Photo check | 30 |
| Daily login | 10 |

High scores (90%+) receive 1.5x XP multiplier.

## Streaks

- Daily activity tracking
- Bonus XP at milestones: 7 days (+100), 30 days (+500), 100 days (+1000)
- Streak resets if a day is missed
