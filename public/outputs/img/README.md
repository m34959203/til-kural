# Папка `img/` — изображения для til-kural-prototype.html

Прототип ссылается на файлы в этой папке. Сохраните 11 сгенерированных изображений сюда под точными именами:

| № | Содержание картинки из чата | Сохранить как |
|---|------------------------------|---------------|
| 1 | Казахская степь на рассвете (юрта, всадник) | `hero-steppe.webp` |
| 2 | Портрет Абая в халате/тымаке | `abai.webp` |
| 3 | Золотой медальон / бейдж с орнаментом | `badge.png` |
| 4 | Горизонтальный орнамент «қошқар мүйіз» | `ornament.webp` |
| 5 | Байтерек / ночная Астана | `astana.webp` |
| 6 | Красно-жёлтые тюльпаны Грейга в степи | `tulips.webp` |
| 7 | Домбра на тёмной поверхности | `dombra.webp` |
| 8 | Интерьер юрты / шаңырақ снизу | `yurt.webp` |
| 9 | Девушка с ноутбуком (альтернативный hero) | `hero-woman.webp` |
| 10 | Портрет Мухтара Ауэзова | `auezov.webp` |
| 11 | Портрет Ахмета Байтурсынова (в очках) | `baitursynov.webp` |

## Как сохранить

1. В чате правый клик по картинке → **Save image as...**
2. Укажите имя из таблицы (с расширением `.webp`; если браузер предлагает `.png/.jpg` — сохраните как есть, а рядом с HTML переименуйте, либо поменяйте расширения в самом HTML).
3. Положите все 11 файлов в эту папку (`img/`).
4. Откройте `til-kural-prototype.html` в браузере — изображения подхватятся автоматически.

## Если расширение не `.webp`

Быстрая замена в HTML через поиск-замену:
- `.webp` → `.png` (или `.jpg`), если сохранили в таком формате.

Либо сконвертируйте в webp через https://squoosh.app (онлайн, бесплатно, без загрузки на сервер).

## Fallback

Если какой-то файл отсутствует, карточка не «ломается» — там стоит `onerror="this.style.display='none'"`, так что показывается исходный градиент или буква-инициал.
